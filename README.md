# Telegram PDF Bot

A professional, production-ready Telegram bot for PDF manipulation with advanced features and monetization.

## Features

### PDF Tools
- Convert images to PDF (JPG, PNG, WEBP)
- Merge multiple PDFs
- Split PDF by page ranges
- Compress PDF with multiple quality levels
- Convert PDF to images
- PDF to Word (DOCX) conversion
- Word to PDF conversion
- Rotate PDF pages
- Add text/image watermarks
- Extract images from PDF
- Password protection/removal
- Page rearrangement/deletion/extraction
- PDF repair
- Page numbering
- Excel/PPT/Text to PDF
- PDF to Text/HTML

### OCR (Premium)
- Text extraction from images
- Scanned PDF text extraction
- Arabic & English OCR support
- Searchable PDF generation

### AI Features (Premium)
- PDF summarization
- Document translation
- Document rewriting
- Text extraction and cleanup
- PDF question-answering
- Document analysis

### Monetization
- Free plan (5 operations/day, 20MB limit)
- Premium Monthly (unlimited, 500MB)
- Premium Yearly (unlimited, 1GB)
- Lifetime access
- Telegram Stars payments
- Referral system
- Advertisement system
